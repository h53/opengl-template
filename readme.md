This is a blank template for OpenGL, with glfw and glad.  
 Thank https://learnopengl.com/ for making the wonderful tutorial.

## how to use
```bash
$ git clone https://github.com/h53/opengl-template.git
$ cd opengl-template
$ mkdir build
$ cd build
$ cmake ..
$ make
$ ./opengl-template
```
